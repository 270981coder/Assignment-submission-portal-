# NovaCSS

NovaCSS is a zero-dependency utility-first CSS framework for React/Vite apps.

## Features
- CSS variables design system
- Light/Dark themes
- Responsive utilities
- Component styles for buttons, cards, forms, alerts, and navbar

## Usage
```css
@import './novacss/novacss.css';
```

## Example
```html
<div class="container py-4">
  <button class="btn btn-primary">Primary</button>
</div>
```
